Scikit-learn

机器学习方法集工具集

提供一批统一化的机器学习方法功能接口，提供聚类、分类、回归、强化学习等计算功能，机器学习最基本且最优秀的 Python 第三方库

官方网站：scikit-learn.org

TensorFlow

AlphaGo 背后的机器学习计算框架

谷歌公司推动的机器学习框架，将数据流图作为基础，图节点代表运算，边代表张量，应用机器学习方法的一种方式，支持谷歌人工智能应用

官方网站：www.tensorflow.org

MXNet

基于神经网络的深度学习计算框架

提供可扩展的神经网络及深度学习计算功能，可用于自动驾驶、机器翻译、语音识别等众多领域，是 Python 最重要的深度学习计算

官方网站：mxnet.incubator.apache.org

Caffe

基于卷积神经网络的深度学习计算框架

提供一批常用的深度学习计算功能，可用于图像分类、图像分割和语音识别等领域，是 Python 最重要的深度学习计算框架

官方网站：caffe.berkeleyvision.org

Theano

基于神经网络的深度学习计算框架

提供一批常用的深度学习计算功能，可用于图像分类、图像分割和语音识别等领域，是 Python 最重要的深度学习计算框架之一

官方网站：deeplearning.net/software/theano

Keras

基于神经网络的深度学习计算框架

提供一批常用的深度学习计算功能，可用于图像分类、图像分割和语音识别等领域，是 Python 最重要的深度学习计算框架之一

官方网站：keras.io

PyTorch

基于神经网络的深度学习计算框架

官方网站：pytorch.org
