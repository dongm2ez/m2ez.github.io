Dongm2ez

我是来自祖国的西北的一个 90 后汉子，在河西走廊的第一站度过了人生中的前十八年，那个地方叫武威，是我的家乡。后来曾经到山东济南求学，又辗转到北京参加工作。

到 2023 年我已经工作八年了，目前在依然在北京工作，主要从事后端技术开发，熟悉 Java 和 Go 语言，对 Python 也有一定了解，同时也使用 PHP 开发一些网站程序。

当前主要关注 Kubernetes、Docker、Istio、Envoy、Service Mesh、Serverless、DevOps 等技术。

业余时间喜欢写作，写过一些技术文章，也写过一些随笔和小说。写作的初衷是为了记录自己的学习和生活，也希望能够帮助到一些人。目前在公众号「河西麦索」上分享文章，欢迎关注。
